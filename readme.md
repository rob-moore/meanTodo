## Installation
In order to run this app on your local machine you'll need to do the following:

1. Clone this repository.
2. `cd` into the `angular training`.
3. Run `npm install`.
4. Run `npm run dev`
5. Open your browser of choice to `http://localhost:8080`.
6. Enjoy and learn.
